 OmniSignal AI

AI-native enterprise intelligence platform that transforms fragmented organizational communication into actionable workflows.

 Features

- Multi-agent workflow orchestration
- Signal extraction
- Priority scoring
- Workflow generation
- Executive summaries
- Organizational memory

 Tech Stack

- Next.js
- FastAPI
- LangGraph
- OpenAI GPT-4.1
- ChromaDB
- TailwindCSS

 How It Works

1. Upload enterprise communication
2. AI extracts critical signals
3. System scores business impact
4. AI generates workflows
5. Reflection agent validates output
6. Executive dashboard displays actions
